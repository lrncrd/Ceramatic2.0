To run the code:

```bash
python process_images.py --model_path "PATH_TO_MODEL.pt" --imgs_dir "PATH_TO_IMGS" --tabular_file "PATH_TO_METADATA.xlsx" --diagnostic --diagnostic_plots --add_bar --install_requirements
```
